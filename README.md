# Sistema Bancario BBVA (Emulación) (Java)

Proyecto de simulación de un sistema bancario en Java que implementa:

- Programación Orientada a Objetos
- Excepciones personalizadas
- Herencia y polimorfismo
- Interfaz Transaccionable
- Manejo de cuentas, depósitos, retiros y transferencias
- Menú interactivo tipo cajero automático

## Tecnologías
- Java SE
- Git & GitHub
